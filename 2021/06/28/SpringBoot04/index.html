Spring Boot 验证器 Hibernate-validator

发表于 2021-06-28 分类于 SpringBoot

Hibernate-validator可实现数据的验证，它是对 JSR ( Java Specification Requests ）标准
的实现 。 在 Web 开发中 ，不需要额外为验证再导入真他依赖，只需要添加 Web 依赖即可。

自定义验证需要提供两个类：①自定义注解类；②自定义验证业务逻辑类。
