git 一

发表于 2021-06-03 分类于 git

Git本地有三个工作区域 【工作目录】【暂存区】【资源库】加上远程的git仓库就可以分为四个工作区域

git add files 从工作目录添加到暂存区

git commit 从暂存区提交到资源库

git push 从本地资源库提交到远程git仓库

git的工作流程一般是这样的：

  1. 在工作目录中添加、修改文件；

  2. 将需要进行版本管理的文件放入暂存区域；git add

  3. 将暂存区域的文件提交到git仓库。git commit

IDEA中集成git

新建项目，绑定git. 将远程的git文件目录拷贝到项目中即可。

git项目下的.git下的config是各项目自己的配置文件，一般配置这个文件不会影响到其它项目。
