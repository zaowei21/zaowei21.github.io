Redis

Redis(Remote Dictionary Server),可基于内存也可持久化的日志型，Key-Value数据库。

持久化（rdb,aof)，效率高，可以用于调整缓存。支持事务。

官网： https://redis.io/

中文网： http://www.redis.cn

配置文件：redis.conf

Redis默认有16个数据库（可以在配置文件里修改），默认使用第0个，可以用select进行切换

五大数据类型

Redis-Key

String

List

Set

Hash

Zset
