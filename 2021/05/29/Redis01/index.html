发表于 2021-05-29 更新于 2021-05-30 分类于 redis

Redis

Redis(Remote Dictionary Server),可基于内存也可持久化的日志型，Key-Value数据库。

持久化（rdb,aof)，效率高，可以用于调整缓存。支持事务。

官网： https://redis.io/

中文网： http://www.redis.cn

配置文件：redis.conf

Redis默认有16个数据库（可以在配置文件里修改），默认使用第0个，可以用select进行切换

五大数据类型

Redis-Key
String
List

列表，基本数据类型。栈，队列

Set

集合，set中的值不能重复。无序不重复集合

Hash

哈希，Map集合，key-Map集合 ，对象用Hash存储更方便。

Zset

有序集合，

三种特殊数据类型

geospatial 地理位置

有六个命令

Hyperloglog

基数（不重复的元素） 内存固定。可以统计不重复元素数量，有误差

Bitmaps

位存储

位图，操作二进制位进行记录，只有0和1两个状态。
